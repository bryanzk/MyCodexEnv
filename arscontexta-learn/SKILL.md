# Ars Contexta Learn

## 概述

用于“引入新知识并融合”：从外部来源学习，并转化为当前领域可消费的结构化笔记。

## 快速流程

1. 明确学习问题与范围。
2. 抽取关键结论与证据。
3. 生成新笔记并链接到现有主题。
4. 标记未决问题供后续 remember/rethink。

## 输出契约

- 新增知识点列表
- 与现有图谱连接点
- 待验证假设

## 常见错误

- 只摘录不融合，形成外来信息孤岛。
- 不标注不确定性，后续难验证。
