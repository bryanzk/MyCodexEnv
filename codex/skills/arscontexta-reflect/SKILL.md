# Ars Contexta Reflect

## 概述

用于执行 6R 中的 Reflect：提升图结构连通性与导航质量，让新旧知识互相可达。

## 快速流程

1. 找出新笔记与孤立笔记（sparse/isolated notes）。
2. 补充双向或多跳关联，优先连接高价值节点。
3. 更新 MOC 的入口与聚类。
4. 记录本轮新增连接数与受影响主题。

## 操作要点

- 优先“解释关系”，不是只加链接。
- 先修复主干主题，再处理边缘主题。
- 对冲突观点建立 tension note 并落在 `ops/tensions/`。

## 常见错误

- 机械加链接不说明关系类型。
- 只改笔记不改 MOC，导致导航断层。
