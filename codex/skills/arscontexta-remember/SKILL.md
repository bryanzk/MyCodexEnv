# Ars Contexta Remember

## 概述

用于沉淀“如何做得更好”的过程记忆，关注错误模式、修正动作与复发预防。

## 快速流程

1. 复盘最近 session 的问题与偏差。
2. 记录触发条件、错误行为、修正动作。
3. 提炼可执行规则写入 ops。
4. 链接到相关任务或笔记。

## 输出契约

- 问题-修正对照表
- 新增规则清单
- 下次检查点

## 常见错误

- 记录事件不记录可执行规则。
- 复盘结论未关联到实际任务。
