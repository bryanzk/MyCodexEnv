FP TX Review Board

这是一页可离线打开的处理面板。它把 72 笔 FP tx 的关键字段、分类假设、EigenTx / Etherscan 链接、选择导出能力，以及“已处理”本地持久化状态放在同一个界面里，适合逐笔过单和批量分发。

Persistent State

已处理状态保存在浏览器 `localStorage`。同一台机器、同一浏览器再次打开页面时会自动恢复。

Export Modes

支持导出“当前选中 tx”的 CSV/JSON，也支持导出“已处理记录” JSON，便于同步给别的同事或归档。

Working Assumption

分类是基于 CSV、RPC 元数据和代表样本复盘得到的族群归类，不等于最终业务真相；页面保留了每笔 tx 的原始关键字段，便于继续人工修正。
