# Ars Contexta Reweave

## 概述

用于执行 6R 中的 Reweave：当新证据出现时，回补历史笔记，避免知识图谱“只增不改”。

## 快速流程

1. 识别被新结论影响的旧笔记。
2. 在旧笔记中新增“更新说明（update rationale）”。
3. 添加到新笔记的链接并保留旧结论上下文。
4. 如变化重大，在 MOC 标注版本变化点。

## 输出契约

- 修改了哪些旧笔记
- 每条变更的原因
- 是否引入新冲突（tension）

## 常见错误

- 直接覆盖历史结论，不保留演化轨迹。
- 只改内容不改连接，导致上下文断裂。
