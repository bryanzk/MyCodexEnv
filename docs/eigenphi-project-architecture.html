阅读结论

业务主链路

MEVAL/MEVScan 处在数据与识别底座；fp-detector 负责质量治理；adaptive-strategy-filter 负责“哪些机会值得看”；eigenphi-cli-arb-mvp 是人工操作面。

辅助分析链路

blocksec-tenderly-parser 和 CallTraceSniffer 更像“解释/诊断层”：前者统一 trace 输入，后者把交易转成 IR、图、报告和 block viewer。
