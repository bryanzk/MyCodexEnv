Delivery Harness Framework / Skill Inventory

DHF 每一步由哪些 Skill 接管。

本页把当前 DHF 工作流程拆成逐步矩阵：每一步使用哪些 Skill、这些 Skill 具体做什么，并附上对应的 GitHub 源码链接。

English version 中文入口 文字路由规范 GitHub 全部 Skills
How to read this

DHF 先路由，再委托给专用 Skill。

生命周期路由器负责恢复持久状态、判断阶段、选择专用工作流。Helper scripts 负责验证证据或 agent plan；本页聚焦 Skill，因为它们定义每个阶段中 agent 应该怎样行动。

Step 01

恢复与路由

从 repo 持久状态启动或恢复复杂任务，然后选择下一步工作流。

delivery-harness-framework

读取 repo 指南、恢复当前阶段、分类生命周期、选择下一步 skill/helper，并明确验证证据门槛。
