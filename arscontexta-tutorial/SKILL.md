# Ars Contexta Tutorial

## 概述

用于一步步带用户完成最小可运行闭环（setup -> reduce -> reflect -> verify -> next），适合首次使用和流程回顾。

## 教学节奏

1. 创建或检查工作区结构。
2. 处理一个真实来源并生成至少一条笔记。
3. 添加至少一条跨笔记连接。
4. 执行一次 verify 检查。
5. 给出下一步动作并解释原因。

## 输出契约

- 当前进度（第 N 步 / 总步数）
- 本步目标
- 本步完成判定

## 常见错误

- 一次性讲完所有概念，不做分步落地。
- 跳过验收判定，无法确认学会。
